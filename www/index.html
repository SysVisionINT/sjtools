The SysVision Java Tools - SJTools is a collection of utility classes, created and used by SysVision on several commercial projects prior to it's release as an Open Source project. 


The main tools in SJTools are:

  • Classloader for loadding classes from the file system and JAR files;
  • Logging API with wrappers for Log4J and JDK 1.4;
  • Generic Cache implementation;
  • Generic Pool implementation;
  • Simple thread pool implementation;
  • Single VM Publish/Subscribe messaging implementation;
  • Task Scheduler;
  • Java reflection and introspection APIs wrapper for bean manipulation;
  • XML-to-Java mapping;
  • DataBase utils;
  • Basic dependency injection implementation.
  • Struts extensions
  • Usefull Enterprise tools
  • Basic Aspect-oriented programming API
  • Mail sending API

SJTools is available in 3 separate JAR files:

  • sjtools.jar - Standalone version
  • sjtools-enterprise.jar - Enterprise edition (standalone and enterprise versions)
  • sjtools-struts.jar - Struts extensions only
